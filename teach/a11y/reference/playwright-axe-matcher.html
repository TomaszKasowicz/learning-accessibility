The toBeAccessible() matcher recipe

Copy-paste scaffold for packaging an axe scan as one reusable Playwright assertion. See lesson 07 for the why.

Matcher contract

return fieldtyperole
passbooleandid the positive assertion succeed? (flip with this.isNot for .not)
message() => stringrendered only on failure / .not-success
namestringoptional; shows in the reporter step

Builder API map (sugar → engine)

AxeBuilder= raw axe optionnote
.withTags([...])runOnly:{type:'tag',values}selects rules by tag
.withRules([...])rules:{ id:{enabled:true} }force-enable specific rules
.options(BAR)full options objectreplaces withTags/withRules — don't mix
.include(sel) / .exclude(sel)contextscope the scan
.disableRules([...])rules:{ id:{enabled:false} }per-call skip (+ TODO/Jira)

Scaffold — a11y-fixtures.ts

// import test + a renamed baseExpect; export your own test + expect
import { test, expect as baseExpect, type Page, type TestInfo } from '@playwright/test';
import AxeBuilder from '@axe-core/playwright';

const BAR = {
  runOnly: { type: 'tag' as const,
    values: ['wcag2a','wcag2aa','wcag21a','wcag21aa','wcag22aa','best-practice'] },
  rules: { 'target-size': { enabled: true } },   // safety net; tag already runs it
};

export const expect = baseExpect.extend({
  async toBeAccessible(
    subject: { page: Page; testInfo: TestInfo },
    options?: { include?: string; disableRules?: string[] },
  ) {
    let b = new AxeBuilder({ page: subject.page }).options(BAR);
    if (options?.include) b = b.include(options.include);
    if (options?.disableRules?.length) b = b.disableRules(options.disableRules);

    const results = await b.analyze();
    const v = results.violations;
    let pass = v.length === 0;

    if (!pass) await subject.testInfo.attach('axe-results.json',
      { body: JSON.stringify(results, null, 2), contentType: 'application/json' });

    const summary = v.map(x =>
      `[${x.impact}] ${x.id} (${x.nodes.length}) → ${x.helpUrl}\n  ` +
      x.nodes.map(n => n.target.join(' ')).join('\n  ')).join('\n\n');

    if (this.isNot) pass = !pass;
    return { pass, name: 'toBeAccessible',
      message: () => `${v.length} violation(s):\n\n${summary}` };
  },
});
export { test };

// TS: make the matcher visible to the type system
declare global {
  namespace PlaywrightTest {
    interface Matchers<R> {
      toBeAccessible(options?: { include?: string; disableRules?: string[] }): Promise<R>;
    }
  }
}

Usage

import { test, expect } from './a11y-fixtures';

test('@a11y page', async ({ page }, testInfo) => {
  await page.goto('/login');
  await expect({ page, testInfo }).toBeAccessible();
  // scoped + temporary skip:
  // await expect({ page, testInfo }).toBeAccessible({ include:'nav', disableRules:['color-contrast'] }); // TODO JIRA-123
});
